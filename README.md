# Led Current Tester

Arduino sketch of a setup use to measure current consumption of WS2812B led

## Setup

- A capacitive touch pad (15 key) with TTP229
- An OLED RGB Screen
- A WS2812B LED strip
- TODO: ACS712 Current Sensors (5A ACS712ELCTR-05B) aka GY-712

## Prerequist Library

- FastLed
- Adafruit GFX
- Adafruit SSD1331
